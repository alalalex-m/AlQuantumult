# 数字金融重点

数字金融**特点**：展业线上化、产品靶向化、业务流程高效化、业务活动广覆盖、服务渠道多元化、风险隐蔽化



数字金融**内容**：以数据资源为关键要素，运用数字技术对金融的业务模式、产品和服务进行创新。数字金融包含数字信贷、数字理财、数字保险等新型金融业态。 传统金融机构、互联网企业与金融科技公司等进入数字金融领域后，扩大了数字金融的从业主体范围。



**发展理念**：创新、协调、绿色、开放、共享



**目标**：

> 第一，全面实现金融业数字化转型。 
>
> 第二，充分释放数据要素潜能。 
>
> 第三，金融服务提质增效显著。 
>
> 第四，构建健全的数字金融治理体系。 
>
> 第五，持续深化关键核心技术应用。 
>
> 第六，建成先进的数字金融基础设施建设。 



**原则**：参照《G20数字普惠金融高级原则》，总结和概括了数字金融发展原则：

> 第一，各国要倡导利用数字技术推动普惠金融发展。
>
> 第二，各国要平衡好数字金融创新与风险。
>
> 第三，各国要构建恰当的数字金融法律与监管框架。 
>
> 第四，各国要扩展数字金融服务生态系统。 
>
> 第五，各国要采取负责任的数字金融措施保护消费者。 
>
> 第六，各国要重视消费者数字技术基础知识和金融知识的普及。 
>
> 第七，各国要促进数字金融服务的客户身份识别。 
>
> 第八，各国要监测数字金融发展进程。



**数字金融与数字普惠金融**：

数字金融更强调数字技术与正规金融相互融合 

数字普惠金融更强调运用数字技术提升金融的包容性



### 数字技术有哪些？核心？

**数字金融中的数字技术**是指可以将数字金融服务中的人、物、场的资金流、信息流全面数字化，改变金融的经营模式、竞争格局和产业生态的技术。

包括人工智能技术、区块链技术、云计算技术、大数据技术和物联网技术等。

**数字技术未来挑战**

1. 引发数据安全问题。如数据泄漏、数据篡改、数据滥用

2. 加剧风险外溢程度。如风险种类更加多样、风险传染范围更广、风险传播速度更快
3. 加剧社会不平等。如技术垄断、数字鸿沟、算法歧视

**发展趋势**

1.推动底层核心技术创新

2.实现金融包容性发展

3.助力建设数字中国



### 什么是数字金融基础设施，主要有哪些，影响因素有哪些？特点？

**数字金融基础设施：**保证数字金融活动正常运行而必需的相关软硬件设施与制度安排。
广义的数字金融基础设施，包括数字金融市场软硬件设施以及数字金融法律法规、会计制度、信息披露原则、社会信用等制度安排，涵盖了数字金融稳定运行所需的各个方面。
狭义的数字金融基础设施，侧重于数字金融市场交易的软硬件设施，尤其是支付清算基础设施和数据平台设施等。



**主要组成部分**：数字金融信息基础设施、数字金融融合基础设施、数字金融创新基础设施、数字金融制度基础设施。

**数据记录收集基础设施**是基石

**数据分析基础设施**是核心

**特点：**去中心化、匿名化、智能化、全球化

数字金融基础设施发展的**影响因素**

| 概念     | 组成部分       |                |                |
| -------- | -------------- | -------------- | -------------- |
| 政策因素 | 规范性政策法规 | 监管性政策法规 | 帮扶性政策法规 |
| 经济因素 | 经济发展水平   | 投资环境       | 金融市场竞争   |
| 技术因素 | 大数据技术     | 云计算技术     | 区块链技术     |

**发展趋势**

1. 更多维度的风险防控体系
2. 更加聚焦普惠金融方向
3. 更趋开放合作与互联互通

### 数字支付是什么

**支付**的概念：社会经济活动中付款人向收款人转移后者可接受的货币债权的行为。

**数字支付定义**：运用自动识别技术、信息通信技术、区块链、大数据、云计算等数字技术，以数字形式实现货币债权转移的创新支付方式。

**特征**：

1. 技术驱动性：数字支付依靠有形的数字支付硬件设施与无形的数字技术实现货币债权的转移。
2. 时空跨越性：数字支付可以不受当前时间节点与地理位置限制，实现货币债权的高效率转移与交易信息的互联交流。
3. 支付方式多样性：数字支付包含更多样的支付方式，能满足消费者对支付方式的差异化需求。

**数字支付**两大核心：支付技术、支付安全

**数字人民币的特点**：无限法偿性、双层流通框架、广义账户体系、可控匿名

**法定数字货币人民币的特点**：由特定运营机构参与运作并向公众兑现，与纸币和硬币具有相等的价值，并作为具有价值属性和法律偿付能力的可控匿名支付工具。

**功能**：提升货币政策的传导效果、提升支付效率、增强支付安全性、降低支付交易成本

**数字信贷：**广义的信贷即信用贷款，指依据借款人的信用发放、以按期偿还本金和利息为条件，不需要提供担保或抵押的贷款，主要包括商业信用贷款和银行信用贷款。

狭义的信贷即为银行信用贷款，专指以银行为主体、以信用为基础的资金借贷行为。

**特点**：1. 以信用为基础，不需要担保或抵押品

1. ​	2.可贷额度较小
2. ​	3.对金融机构的风险管理能力要求较高

特点3：广覆盖

**数字信贷发展的意义**

（1）数字信贷为解决小微信贷“不可能三角”的困境提供了新的思路，成为助力普惠金融发展的重要方式

（2）数字信贷有助于推动金融行业的数字化转型，提升数字金融发展水平，助推我国数字金融生态建设

（3）数字信贷从消费升级和产业转型两个层面推动实体经济高质量发展

**数字理财定义：**是指为传统金融机构、互联网企业以及金融科技公司等运用数字技术为客户提供互动式和个性化的创新型理财产品及服务的新型数字金融业态。

**特点**：可获得性高、服务个性化、更加注重投资者教育

**现实价值**：

第一，数字理财改变传统理财对居民消费的影响路径，促进居民消费升级。

第二，数字理财通过改变数字理财供给方的成本费用结构以及促进多主体间竞争合作，优化供给方的资源配置。

第三，数字理财缓解了人们日益增长的金融服务需求和金融供给不平衡不充分之间的矛盾，促进普惠金融发展。

第四，数字理财拉动内需并通过金融市场投资实体企业，促进经济增长。

**发展趋势**

未来数字理财的发展将继续立足长尾客户市场，通过构建数字理财生态圈，将理财融入居民生活场景中，为客户提供一体化的数字理财服务。

第一，数字理财继续发力投资者教育，提高投资者金融素养。

第二，数字理财将通过技术与人工结合，提高客户理解度。

第三，数字理财将通过多主体间的协作，推动构建生态圈。 

### 产业数字金融是什么？

以数据为重要生产要素，利用人工智能、大数据、云计算、区块链、物联网等数字技术，使用数字化手段为特定产业提供投融资、支付结算、财务管理、信用评估、国际业务等综合金融服务，促进产业转型升级的新型金融业态。

**特点**：综合性、多层次性、全流程性、政策导向性

**产业数字金融与数字供应链金融的区别与联系**

区别：

| 角度       | 产业数字金融                                                 | 数字供应链金融                                               |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 服务对象   | 产业生态                                                     | 围绕核心企业，服务上下游企业                                 |
| 服务种类   | 政府投资基金、政策性融资担保、银行信贷、多层次资本市场、动产融资、租赁、保理等 | 动产融资、租赁、保理等                                       |
| 服务阶段   | 产业各生命周期                                               | 成熟期、拥有完善供应链                                       |
| 市场化程度 | 政府参与度高，在产业政策指导下服务特定产业，根据产业政策调整发展战略 | 多由金融机构、核心企业或者第三方平台发起，其发展由市场决定，受产业政策影响较小 |
| 理论基础   | 产融结合理论、产业生命周期理论、产业演化理论与最优金融结构理论等 | 复杂网络理论、融资契约理论、供应链管理理论等                 |
